# NES ROMs Directory

Please place your legal .nes ROM files in this directory.
The filenames should match the `romUrl` paths defined in `src/data/nesGames.ts`.

Example mapping:
- Super Mario Bros. -> `super_mario_bros.nes`
- Contra -> `contra.nes`
- The Legend of Zelda -> `zelda.nes`
- etc.

Ensure you have the legal right to use these ROMs.
