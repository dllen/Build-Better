About This Tool

Fügen Sie Ihren PDFs professionelle Seitenzahlen hinzu. Wählen Sie aus verschiedenen Formaten, Positionen und Stilen, um das Layout Ihres Dokuments zu perfektionieren.

How to Use

  1. 1

    PDF hochladen

    Laden Sie das Dokument hoch, das nummeriert werden soll.

  2. 2

    Formatierung wählen

    Bestimmen Sie Position (z.B. unten rechts), Startnummer und das Zahlenformat.

  3. 3

    Anwenden

    Fügen Sie die Paginierung hinzu und laden Sie das Ergebnis herunter.

Frequently Asked Questions

Kann ich die erste Seite auslassen?

Ja, Sie können festlegen, dass die Nummerierung erst ab einer bestimmten Seite (z.B. nach dem Deckblatt) beginnt.

Welche Formate gibt es?

Arabische Zahlen (1, 2, 3), römische Zahlen (i, ii, iii) oder das Format "Seite X von Y".

Kann ich die Schriftart ändern?

Ja, Sie können Schriftgröße und Typ an Ihr Dokument anpassen.
